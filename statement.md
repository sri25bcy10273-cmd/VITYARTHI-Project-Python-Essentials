Problem Statement
his project addresses these challenges by providing a digital solution that automates medicine inventory management, reduces human errors, and improves operational efficiency for small pharmacies or medical stores.

Scope of Project -
The scope of this project is limited to a console-based application that manages the core inventory operations of a pharmacy. It focuses on the following functionalities:
Medicine Management: Add, view, search, update, and delete medicine records.
Inventory Tracking: Store and display medicine details (name, price, quantity, expiry date).
Database Integration: Use SQLite for persistent storage of medicine records.
User Interaction: Provide a simple menu-driven interface for pharmacists or store staff to interact with the system.
The project does not include advanced features like billing, sales tracking, user authentication, or integration with external systems. It is designed for small-scale use and does not support multi-user access or complex reporting

Target Users -
i) Pharmasists
ii) Pharmacy Staff
iii) Small Pharmacy Owners

High Level Features -
1) Add Medicine
2) Delete Medicine
3) Search Medicine
4) View Inventory
5) Update Medicine
6) Database Initialization
7) Input Validation
8) Menu Driven Interface
