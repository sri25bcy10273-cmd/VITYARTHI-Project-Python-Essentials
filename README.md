# Python-Essentials-VITYARTHI-Project
Pharmacy Management CRUS(Create, Read, Update, Delete).
 The system stores medicine details including name, price, quantity, and expiry date.
 Features:
Add new medicine to inventory
View current medicine inventory
Search medicines by name using partial match
Update medicine price and quantity by ID
Delete medicine by ID with confirmation
Tools Used:
Python (core language)
SQLite3 (database for storage)
Steps to Install and Run the Project:
Ensure Python is installed.
No external dependencies required other than standard sqlite3 module.
Run the script PharmarCRUD.py directly with Python interpreter.
Instructions for Testing:
Run the script.
Use the displayed menu options 1 to 6 to test adding, viewing, searching, updating, and deleting medicines.
Input data as prompted (medicine details, IDs, etc.).
Confirm deletions when prompted.
Verify changes by viewing inventory or searching.
